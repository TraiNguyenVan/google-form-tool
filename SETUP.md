# Setup Instructions

## Prerequisites
- **Browser**: Google Chrome (version 88+ required for Manifest V3).
- **API Key**: A valid Gemini API key (from Google AI Studio).

## Installation
1. Clone or download this repository.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click **Load unpacked** and select the project directory.

## Configuration
1. Open the extension's Side Panel.
2. Navigate to the `My Data` tab.
3. Enter your Gemini API key to enable AI-powered Smart Fill.

## Development & Testing
- **Dependencies**: None. This is a vanilla JavaScript Chrome extension. No build steps (like Webpack or npm) are required.
- **Testing**: Testing is currently done manually. Load the unpacked extension and verify against live Google Forms.
- **Deployment**: Distribution is currently via manual unpacked loading. Not yet published to the Chrome Web Store.
